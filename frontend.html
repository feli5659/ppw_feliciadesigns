front end design

game design

The game is based on the historical drama hit-series “Outlander” and therefore takes place in the Scottish Highlands in the 18th century.

The aim of the game is saving Jamie and his family from the evil “Redcoat” soldiers.

Captain Jack Randall, the notorious barbarian soldier, has a personal vendetta against Jamie and therefore gives more points in this game.

project description

This project was the first time we were introduced to JavaScript. Through building this game and understanding the game mechanics, I gained a very basic knowledge of JavaScript, that enabled me to not only build this game, but also understand the functionality of many other features in webdesign in general.

There was also an extensive design process prior to coding the game, where we gained knowledge in colour theory, using Adobe Illustrator and creating perspective in a design.
